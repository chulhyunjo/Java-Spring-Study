# 3-2. 불 (boolean)

---

## 불 연산

---

```java
// boolean sample code
2 > 1            // true
1 == 2           // false
3 % 2 == 1       // true
"3".equals("2")  // false
```



## 조건문

---

```java
// sample code
public class Sample {
    public static void main(String[] args) {
        int base = 180;
        int height = 185;
        boolean isTall = height > base;

        if (isTall) {
            System.out.println("키가 큽니다.");
        }
    }
}

// output
키가 큽니다.
```

```java
// sample code
public class Sample {
    public static void main(String[] args) {
        int i = 3;
        boolean isOdd = i % 2 == 1;
        System.out.println(isOdd);
    }
}
```
